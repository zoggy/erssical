Compiling Erssical requires the following software to be installed:

  • OCaml >= 4.00,
  • Findlib <= 1.3.3 (should work with priori versions),
  • OCamlnet >= 3.6.3 (with Nethttpd library enabled),
  • OCaml-RSS >= 2.2.0,
  • OCurl >= 0.5.6 (not tested with prior versions),
  • Xtmpl >= 0.6,
  • [optional] Stog >= 0.8.0 for the documentation.

After extracting the source tree, go to the extracted directory and type:

make all make install # with sufficient rights

This will install erssical[.byte] and erssical-http[.byte] in the same directory as ocamlc, and the erssical library with OCamlfind.
