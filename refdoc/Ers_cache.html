Module Ers_cache

module Ers_cache: sig .. end
Caching fetched RSS channels.
type t
a directory
val mk_cache : string -> t
Create a cache from the given directory name. Raise Failure if the cache could not be created.
val print_error : (string -> unit) Pervasives.ref
The function called to print error messages. Default is prerr_endline.
val default_ttl : int Pervasives.ref
Default time to live of cached RSS channels, in minutes. Used when a cached RSS channel does no specify a ttl. Default is 60 minutes.
val get : t -> Neturl.url -> string
Fetch the given URL of RSS channel, first looking in cache and fetching only if time to live has expired.
